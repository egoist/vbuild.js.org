ESLint

You can use option eslint to run eslint during compilation, it defaults to use eslint-config-rem which based on eslint-config-xo but customized for Vue apps.

Most likely you only need eslint while building for production, since it might kill the process in development mode.

{
  // add as `production` speicifed options
  // or you can use different config files for different builds
  production: {
    // simply use default config
    // eslint: true
    // __or__
    eslint: {
      // use a desired config
      configFile: require.resolve('eslint-config-airbnb')
    }
  }
}

You can pass all eslint options here.

Gzip

Prepare compressed versions of assets to serve them with Content-Encoding. See Usage.

module.exports = {
  production: {
    gzip: true,
    // or with options
    // gzip: {}
  }
}
